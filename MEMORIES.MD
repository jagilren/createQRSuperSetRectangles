# PARA GENERAR TCARDS
Generar el archivo de TAGS.XLSX
Borrar el contenido del folder URL
Ejecutar main.py 
Esto Genera un archivo de Word con todos los TAG en formato TCARD


# PARA GENERAR ADHESIVOS
Generar el archivo  TAGS.XLSX
Borrar el contenido del folder URL
Ejecutar main_blowers.py 
Esto Genera un archivo de Microsoft  Word con todos los TAG en formato ADHESIVE
